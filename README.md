# twitch-youtube-bot -

A bot that listens for YouTube requests in Twitch chat, and displays them on screen for the streamer to view.

## Built Using - 
```
Node.js
Socket.IO
tmi.js
React
ReactSocketIOHooks
```


## Try the Bot -
[LIVE on Heroku](https://twitch-youtube-bot.herokuapp.com/)
